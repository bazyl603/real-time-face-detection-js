# real-time-face-detection-js

### This is a simple face detection for recognising humour
### This projection was used to teach :
- Streaming a webcam through HTML
- Using Face API to detect faces in real time
- Drawing facial landmarks in real time
- Determining emotion through facial expressions in real time

More about api: https://github.com/justadudewhohacks/face-api.js.git

## Operation and use of this application
The application uses the camera image for API analysis. It then displays landmarks on your face in real time and emotion.

### Launching the application:
The application must be started by a live server.
For example, an extension in visual studio code.
